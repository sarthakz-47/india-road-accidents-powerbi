# India Road Accidents & Safety Dashboard
## Business Intelligence Lab Mini Project

**Tool:** Microsoft Power BI
**Dataset:** India Road Accident Dataset - Predictive Analysis (Kaggle)
**Subject:** Business Intelligence Lab (ITL601)

---

## Project Overview
This dashboard analyzes 4.5 lakh+ road accidents across India to help
traffic authorities make data-driven safety decisions.

## Dashboard Pages
- **Page 1 - Overview:** KPI cards, year trends, severity breakdown
- **Page 2 - Location Analysis:** India state map, top dangerous states
- **Page 3 - Cause Analysis:** Weather, vehicle type, alcohol, driver age
- **Page 4 - BI Decisions:** Risk heatmap, policy recommendations

## Dataset Columns (21 features)
State, City, Year, Month, Day, Time, Severity, Vehicles Involved,
Vehicle Type, Casualties, Fatalities, Weather, Road Type,
Road Condition, Lighting, Traffic Control, Speed Limit,
Driver Age, Driver Gender, License Status, Alcohol Involvement

## Key BI Decisions Derived
1. High fatality states need immediate highway patrol enforcement
2. Night-time accidents = highest severity, recommend highway lighting
3. Alcohol accidents are 3x more fatal, enforce weekend checkpoints
4. Unlicensed drivers = major risk factor, need stricter RTO checks
